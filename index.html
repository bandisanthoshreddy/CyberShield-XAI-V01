How It Works

01
Language detection + normalisation

Input is identified as EN/HI/TE and normalised using IndicNLP for Devanagari and Telugu scripts, handling code-mixing.

02
MuRIL transformer

Google's Multilingual Representations for Indian Languages model, fine-tuned on labelled cyberbullying data across all three languages with Focal Loss.

03
XAI explanations

Attention heatmaps, LIME local perturbations, and SHAP Shapley values highlight exactly why the model flagged the text.

04
Prevention actions

Based on label and confidence, the system recommends: warn the poster, flag for review, or auto-remove high-severity content.

🔬

Model architecture

MuRIL-base-cased (12-layer BERT) fine-tuned with Focal Loss (γ=2.0) and class weights across 5 cyberbullying categories.

🌐

Multilingual support

English, Hindi (Devanagari + Romanized), Telugu (native + code-mixed Tenglish) — single unified model.

💡

Explainability

SHAP (exact attribution), LIME (local perturbations), and Transformer Attention maps — three perspectives on every decision.

🛡️

Prevention layer

REST API enables real-time integration with social media platforms to flag, warn, or block harmful content before it reaches victims.
